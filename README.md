## 🏢 Vitrine de Projetos em Destaque

### 🏛️ [RN Rebuilder](https://github.com/seu-usuario/rn-rebuilder)
> **Projeto Acadêmico (A3) — Orientador: Prof. Erivelton Lima**
> 
> Plataforma de utilidade pública desenvolvida para estreitar a comunicação entre a população e a administração governamental, permitindo o reporte ágil e simplificado de problemas na infraestrutura urbana do Rio Grande do Norte.
> 
> * **Acessibilidade Absoluta:** Relação de interface intuitiva desenhada para usuários de qualquer faixa etária.
> * **Fluxo de Autenticação:** Módulos enxutos de Login e Registro de Contas com validação de credenciais.
> * **Painel Administrativo Interno:** Dashboard restrito (`Admin / Admin`) para triagem, monitoramento e atualização de status dos incidentes em tempo real (*Pendente*, *Em Andamento*, *Concluído*).

### 🎲 [PDL A 0 - Roguelike Draft Simulator](https://github.com/seu-usuario/pdl-a-0) *(WIP)*
> Simulador tático de futebol com mecânicas de Roguelike inspirado no sistema do clássico site *7a0*. 
> 
> * Montagem de elencos baseada em sorteios aleatórios restritos por posições táticas do campo.
> * Lógica procedural de cálculo de gols, faltas e interrupções por VAR baseada em diferença de força estatística ($OVR$).
> * Botão especial de invocação "Super Trunfo" para puxar o elenco lendário do time PDL.
